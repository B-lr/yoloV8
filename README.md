# YOLOv8 Object Detection on COCO 2017 Validation Dataset
This project demonstrates how to run object detection using YOLOv8 on the COCO 2017 validation dataset (5,000 images). It includes steps for setup, dataset download, running inference, and visualizing results before and after prediction.
# Features
Run object detection on COCO 2017 validation images
Display input images before prediction
Save detection outputs with bounding boxes
Option to compare original vs predicted images
Works on CPU or GPU
# Dataset
We use the COCO 2017 validation dataset:
~5,000 images
